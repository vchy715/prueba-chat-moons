# prueba-chat-moons

Aplicación de chat en tiempo real usando React, Node.js y Socket.io.

## Levantar aplicación web

### Cliente
* Navegar a la carpeta "client"
* Abrir terminal y correr los siguietes comandos:
  * npm install
  * npm start

### Servidor
* Navegar a la carpeta "server"
* Abrir terminal y correr los siguietes comandos:
  * npm install
  * npm start
